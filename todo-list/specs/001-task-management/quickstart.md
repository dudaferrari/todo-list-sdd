# Quickstart

## Prerequisites

- Node.js installed locally
- The backend app can be started from the repository

## Validation Steps

1. Start the backend server.
2. Send a POST request to `/tasks` with title, description, reminderDate, and status.
3. Send a GET request to `/tasks` and confirm the task appears.
4. Send a DELETE request to `/tasks/{id}` and confirm the task is removed.
