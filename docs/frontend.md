# Frontend

O frontend foi implementado com React para oferecer uma interface simples, clara e interativa.

Ele permite:

- preencher um formulário com os dados da tarefa;
- visualizar a lista de tarefas cadastradas;
- excluir tarefas da lista;
- informar lembretes associados às tarefas.

## Diretrizes de experiência e usabilidade

A interface foi pensada para ser objetiva e fácil de usar, respeitando a constituição do projeto. O foco é reduzir atritos, manter o fluxo simples e garantir que as operações principais sejam intuitivas para o usuário.
