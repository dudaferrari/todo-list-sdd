# Features

As funcionalidades principais da aplicação são:

- cadastro de tarefas;
- visualização de todas as tarefas;
- exclusão de tarefas;
- cadastro de lembretes associados às tarefas;
- armazenamento em memória;
- interface simples para interação.

## Qualidade percebida pelo usuário

Essas funções foram implementadas com foco em simplicidade, confiabilidade e facilidade de uso, mantendo o escopo alinhado à constituição do projeto.
